# DenoisingAndEnhacement
This project contains cpu and gpu(cuda) code:
Cpu side:
  => Automatic brightness and constrast
  => Bad Pixel Replacement
  => Defog on Image
  => Haze Remove on Image
  => Nuc(Non Uniform Correction)
  => Contrast Stretching
Cuda side:
  => Bad Pixel Correction
  => Dehaze
      -> Box Filter
      -> Dark Channel
      -> Minimum Filter
      -> Transpose
  => Radix Sort
  => Contrast Stretching
